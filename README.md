# TZ Sync 🌍

A simple, beautiful multi-timezone meeting scheduler. See 3 timezones at a glance on mobile and desktop.

![TZ Sync](https://via.placeholder.com/800x400/1e293b/ffffff?text=TZ+Sync)

## Features

- 🕐 **3 Timezone Views** - See your timezone, HK/Singapore, and CET side by side
- 📱 **Responsive Design** - Works beautifully on mobile and desktop
- 🎨 **Visual Timeline** - 24-hour timeline showing work hour overlaps
- 🟢 **Work Hours Indicator** - Green highlighting for 9AM-6PM in each zone
- ⚡ **Real-time Updates** - Clocks update every second
- 💾 **Persistent Settings** - Your timezone preferences are saved locally

## Usage

1. Open the app
2. Your timezone is auto-detected
3. See Singapore/HK and CET times at a glance
4. Use the timeline to find overlapping work hours
5. Check "Quick Reference" for what time it is elsewhere when it's 9AM/noon/3PM/6PM for you

## Configuration

Click the ⚙️ settings icon to customize timezones.

Default setup:
- **Your Timezone** - Auto-detected
- **Timezone 2** - Singapore/Hong Kong (GMT+8)
- **Timezone 3** - CET (Central European Time)

## Deployment

This app is deployed on Vercel. Any push to `main` branch triggers automatic deployment.

## Tech Stack

- HTML5 + Tailwind CSS (via CDN)
- Vanilla JavaScript (no framework needed)
- Deployed on Vercel
